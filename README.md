## Getting Started

### Prerequisites

- NPM and NodeJS
- PM2
- Adonis JS
- MySQL

### Installing

### NPM & AdonisJS

```
brew install npm && brew install node
```

### AdonisJS

```
npm i -g @adonisjs/cli
```

### PM2

```
npm i -g pm2
```

### MySQL

```
brew install mysql
```

## Running the scheduler

Directly run PM2 or run scheduler by

```
node ace run:scheduler
```

## Running the test

```
adonis test
```
